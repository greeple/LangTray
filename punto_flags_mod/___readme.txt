
 1. скопировать punto.exe из папки с установленонй программой рядом с ___mod_flags_icons.bat
 2. скопировать необходимы для замены иконки флагов в папку flags
    /названия файлов должны соответствовать находящимся в папке иконкам/
    можно использовать иконки одного из 13 приложенных вариантов (копировать в корень flags)
 3. запустите ___mod_flags_icons.bat
 4. закрыть Punto Switcher и вернуть модифицированный файл в папку с программой
